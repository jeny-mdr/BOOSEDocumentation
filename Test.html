Class AppCanvasTests
Contains unit tests for the AppCanvas class and its related BOOSE commands.
Syntax
C#
    public class AppCanvasTests
Remarks
This test class validates the core functionality of the AppCanvas, including: setting pen colors, moving the cursor, drawing shapes, and running multiline programs. It ensures that all commands execute correctly and modify the canvas as expected.

Each test method follows the Arrange-Act-Assert pattern: Arrange: Set up the necessary canvas and objects. Act: Execute the command or operation being tested. Assert: Verify that the canvas or object state matches expected results.

This class also includes helper methods, such as CreateCanvas, which provide a convenient way to create a canvas for testing both constructors.

Class MyWrite
Represents a BOOSE command that writes text to the canvas.
Syntax
C#
    public class MyWrite : Command
Remarks
The WRITE command accepts one or more string parameters, joins them into a single string, trims surrounding quotes, and outputs the resulting text to the canvas.

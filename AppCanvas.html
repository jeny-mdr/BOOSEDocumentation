Method Set
Resizes the canvas to the specified width and height, and resets its contents and cursor position.
Syntax
C#
    public void Set(int xsize, int ysize)
Parameters
xsize
The new width of the canvas in pixels.
ysize
The new height of the canvas in pixels.
Remarks
This method creates new bitmaps for the canvas and cursor, reinitializes the graphics objects, sets the pen and brush to default black, and calls Reset to clear the canvas.

Method SetBitmap
Replaces the internal canvas bitmap with a new bitmap and updates the graphics object.
Syntax
C#
    public void SetBitmap(Bitmap newBmp)
Parameters
newBmp
The new bitmap to set. If null, the method does nothing.
Remarks
This method clones the provided bitmap to avoid modifying the original image. It also reinitializes the graphics object to allow further drawing operations on the new bitmap.
